# llm-agents

Series of LLM Agents to perform tasks such as:

- Curated web search
- Autonomous web scraping
- More to come...

### LLM Models supported
- Gemini-1.5-Flash
- Gemini-1.5-Pro
- Claude-3.5-Sonnet
- Claude-3-Opus
